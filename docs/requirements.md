\## 1. Overview

TuneFlex is an offline AI-based music application with two core components:

1\. Image-to-Song (Search existing songs)

2\. Text-to-Sing (Generate new singing audio)



\## 2. Functional Requirements



\### Image-to-Song

\- User uploads an image

\- System detects meaningful objects and text from the image

\- Extracted meaning is matched with existing songs

\- Best matching song is played



\### Text-to-Sing

\- User enters text/lyrics

\- System generates a new singing voice

\- Generated audio is played to the user



\## 3. Non-Functional Requirements

\- Works without cloud dependency

\- Supports multiple languages in image text detection

\- Fast response time

\- Modular and scalable architecture

\- Runs on desktop and mobile environments
